# Simultion library for Maplestory

- 메이플스토리에 관련된 다양한 연산을 지원하는 패키지.
- Package for various calculation related with Maplestory.

## Install
- `pip install simaple`

## Documentation
- https://simaple.readthedocs.io/en/latest/

## Functions

### 아이템 관련

- 강화 수치 계산, 강화 수치 역연산
- gear improvement calculation / derivation

### 스텟 관련

- 스텟 공격력 계산, 환산 주스텟 계산

### 홈페이지 연동

- 홈페이지 데이터 불러오기 기능
